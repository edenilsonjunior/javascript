Bem vindo ao IFilmes!

Seja bem-vindo ao IFilmes! Aqui, você encontrará uma vasta seleção de entretenimento para todas as preferências. Por trás deste portal, está um trabalho de faculdade que se transformou em um espaço de referência para amantes do cinema. Utilizando JavaScript e o framework Bootstrap, juntamos o útil ao agradável, permitindo o consumo de APIs para disponibilizar os filmes.
